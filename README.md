# Felinator-
Felinator é um simulador interativo de genética felina, projetado para ensinar cruzamentos, probabilidades e padrões de pelagem de forma lúdica e visual. Com modos para alunos e professores, o jogo combina ciência real, probabilidades genéticas, quadrado de Punnett e recursos didáticos avançados — tudo diretamente no navegador.
